Global Health Data Analysis – SQL Project
Overview

This repository contains SQL queries used to analyze global health data across three major diseases:

Malaria
HIV/AIDS
Tuberculosis (TB)

The goal of this project is to transform raw datasets into actionable insights, supporting data-driven decision-making in global health.

Objectives
Analyze global disease burden across continents and countries
Identify trends and patterns over time
Compare regional performance and disparities
Support dashboard development (Power BI / Tableau)
Generate insights for policy and intervention strategies

Project Structure
data > raw datasets > queries (data_cleaning, sql aggregation_queries.sql, kpi_calculations.sql, trend_analysis.sql, regional_analysis.sql, outputs/
query results / exported table to github and Powerbi for Visualization).

Key SQL Analyses
1. Data Cleaning & Preparation
Handling missing values
Standardizing country and region names
Filtering relevant indicators

2. KPI Calculations
Total cases by disease
Year-over-Year (YoY) growth rates
Disease-specific aggregations

3. Trend Analysis
Time-series analysis of disease spread
Identification of peaks and declines
Comparative trends across diseases

4. Regional & Continental Analysis
Disease burden by continent
Country-level comparisons
Identification of high-risk regions

5. Advanced Queries
Case distribution ranking
Contribution analysis (% share by region)
Multi-disease comparative analysis

Sample Insights
Africa carries the highest burden of malaria, HIV/AIDS, and TB
Nigeria ranks among the highest contributors to malaria cases
Malaria remains the dominant global disease burden
Disease distribution is highly unequal across regions
